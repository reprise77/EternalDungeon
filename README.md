# EternalDungeon
Проект PYTHON
План: EternalDungeon.
Команда: Земсков Семён, Ли Арсений.
Описание проекта:
Roguelike игра с использованием библиотеки pygame.
Сеттинг: средневековые подземелья с элементами фентези
Срок выполнения: 1 месяц
